# Dungeon Game

Среда разработки: Unity 2022.3.12f, Universal RP

## Правила игры: 
Игра начинается в лобби. При наведении на дверь и клик переходим на уровень подземелья.
На уровне подземелья, необходимо победить всех врагов. После этого можно вернуться в лобби и забрать награду. 
Или пойти на следующий уровень и добыть более ценную награду. 
После сбора награды происходит обнуление прогресса.

- нажатие ЛКМ на свободную клетку - перемещение ГГ, если это возможно.
